About This Project

This Pokédex is a fan-made web app built to explore Pokémon types, stats, and evolutions using data from PokéAPI.

GitHub Repository

github.com/apirollijr/pokedex

Developer Bio

Hi! I’m Anthony Pirolli Jr, a developer passionate about front-end design, JavaScript, and APIs. I built this Pokédex as part of my journey to become a full-stack web developer.

Tech Stack

  • HTML5 – Semantic structure and accessibility
  • SCSS – Modular and maintainable styling
  • JavaScript – Dynamic loading, filtering, and interaction
  • PokéAPI – Live Pokémon data including types, stats, and evolutions
